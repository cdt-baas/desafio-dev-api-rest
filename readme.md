# Dock API

## Como executar?
Instalar os pacotes de dependencia
```
pip install -r requirements.txt
python run.py
```

Após a primeira execução do run.py, é necessário comentar as linhas 4 e 5 do run.py:

    from config import database
    database.init_db()
